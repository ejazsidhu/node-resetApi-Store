1- user "npm init" to set up package.json folder for dependancy managment
2- install "npm install --save express" for handling routing for API
3- install "npm install --save-dev nodemon" this is handling development changes and restarting server automaticaly.
4- install "npm install --save morgan" for logging error of app or api call related info in console
5- install "npm install --save body-parser" helps to pars ulr and body but does not support html
